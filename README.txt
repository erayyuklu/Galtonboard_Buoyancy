# Galton Board with Buoyancy Simulation

A physics simulation demonstrating the combination of binomial distribution and buoyancy principles using the Manim animation framework. This project creates a unique visualization where particles rise through a Galton board due to buoyancy forces in a fluid medium.

## Requirements

### Critical Version Requirements
- Python version <= 3.10
- MiKTeX (latest version)
- Manim version 0.17.3

### Dependencies
```bash
pip install manim==0.17.3
pip install numpy
```

### Running the Code
Run the command:
```bash
manim -pql main.py GaltonBoardBuoyancy
```
and the video will be created with the name GaltonBoardBuoyancy.mp4 in the "galton_buoancy\media\videos\main\480p15" directory
